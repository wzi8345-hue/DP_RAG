# 对比分析 skill 示例（供调试 / 提示词参考，不直接进入运行链路）

这些示例用于人工校准 triggers 与 plan/policy 提示词的效果，可在迭代时参考；
运行时只有 SKILL.md / plan.md / policy.md / synthesis.md 会被加载器读取。

## 命中示例（应选 comparison）
- "锌铝镁镀层和纯锌镀层的耐蚀性能对比" → 对象=[锌铝镁镀层, 纯锌镀层]，维度=[耐蚀性能]
- "低温法和高温法合成 MoS2 有什么区别" → 对象=[低温合成, 高温合成]，维度=[工艺, 产物特性, 机理]
- "几种锈层原位表征手段各自的优劣" → 对象=[各表征手段]，维度=[原理, 适用性, 局限]

## 不命中示例（应交给别的 skill）
- "锌铝镁镀层的耐蚀机理" → mechanism_analysis（单对象深挖机理，无比较关系）
- "综述一下耐候钢的研究现状" → literature_review（面上梳理）

## 对称拆解示范（对象 A=锌铝镁, B=纯锌；维度=耐蚀/机理/应用）
facets:
- corrosion_A: 锌铝镁镀层的耐蚀性能与速率
- corrosion_B: 纯锌镀层的耐蚀性能与速率
- mechanism_A: 锌铝镁镀层的耐蚀机理与腐蚀产物
- mechanism_B: 纯锌镀层的耐蚀机理与腐蚀产物
